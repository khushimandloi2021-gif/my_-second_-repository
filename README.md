# Treasure Hunt Game 🎁

A fun terminal-based game built with Python!

## How to play
- 9 boxes shown in 3x3 grid
- Treasure hidden randomly in one box
- You get 3 chances to find it
- Enter box number like 11, 23, 32

## Concepts used
- Random module
- 2D Lists
- sys.exit()
- User input handling
